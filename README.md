# Console Projectile Game (C++)
A level-based console game written in C++ that focuses on core programming concepts such as game logic, state management, simple physics simulation (projectile motion and gravity), collision detection, and Win/Lose conditions.

## Features
- Projectile system with gravity and angle control
- Limited shots per level
- Enemy collision detection
- Grid-based level design
- Win/Lose system for each level
